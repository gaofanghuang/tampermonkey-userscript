# tampermonkey-userscript
油猴插件脚本

---

## 知乎Plus:

### 功能说明：

  1. 暗色极简阅读模式；

  2. 去除官方或用户插入的广告;

  3. 纤细优雅的滚动条。

### 插件下载地址
